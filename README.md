# 🧮 Calculadora Científica Pro — JS

![JS](https://img.shields.io/badge/JavaScript-ES6+-yellow?style=for-the-badge&logo=javascript)
![CSS](https://img.shields.io/badge/Design-Glassmorphism-blue?style=for-the-badge)

Una calculadora científica avanzada desarrollada con **JavaScript Vanila**, enfocada en la experiencia de usuario y la precisión matemática.

## 🌟 Características Destacadas
- **Modos Duales:** Interfaz conmutable entre modo **Simple** y **Científico**.
- **Gestión de Memoria:** Historial de operaciones persistente (usando `localStorage`).
- **Exportación de Datos:** Capacidad para exportar el historial de cálculos a un archivo de texto (.txt).
- **Trigonometría Flexible:** Soporte para cálculos en Radianes (RAD) y Grados (°).
- **Sistema de Tests:** Incluye un panel de pruebas unitarias integrado para validar la lógica matemática.
- **UI Adaptativa:** Modo oscuro/claro automático y diseño responsivo.

## 🛠️ Stack Tecnológico
- **HTML5** (Estructura semántica)
- **CSS3** (Gradients, Grid Layout, Animaciones de transición)
- **JavaScript** (Lógica de evaluación, API de Blobs para exportación, Manejo de errores)

## 🧪 Pruebas (Testing)
El proyecto incluye una función `runTests()` que valida operaciones críticas como:
- Jerarquía de operaciones: `(2+3)*4`
- Potencias y raíces.
- Manejo de errores en logaritmos y divisiones por cero.

## 🚀 Instalación y Despliegue
1. Clona el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
3. ¡O actívalo en **GitHub Pages** para usarlo desde tu móvil!
